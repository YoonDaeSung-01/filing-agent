## 집 노트북에서 이어서 하기
전체 환경 세팅은 `docs/SETUP.md`에 있다. 두 노트북 모두 이미 세팅돼 있다면 아래 3단계로 이어받는다.

```powershell
git pull                          # 최신 코드 받기
uv sync                           # 새 의존성(langfuse v4 등) 반영
docker compose up -d pgvector     # DB만 띄운다(서비스명 꼭 붙일 것 — Phase 6에서 app 추가됨)
```

- 코드만 빠르게 확인하려면(키·DB 불필요): `uv run pytest` → 102개 통과.
- RAG·에이전트를 실제로 돌리려면: `.env`에 키가 있어야 하고(노트북마다 직접), `uv run python scripts/ingest_all.py`로
  데이터를 재적재한다. `data/`·pgvector 내용은 git에 없어 노트북마다 1회 재생성한다.
- `.env`에 Langfuse 키는 선택. 없으면 트레이싱만 꺼지고 앱은 정상 동작한다.
